react progress bar

# Install 
```
  yarn add @ohaeseong/react-progress-bar
```

# Usage

```
import React from 'react';
import { ProgressBar } from '@ohaeseong/react-progress-bar';

function App() {

  return <ProgressBar value={80} />;
}

```
